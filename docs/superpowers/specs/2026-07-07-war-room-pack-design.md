# War Room Pack Design

## Goal
Create an all-in-one but balanced war-room pack for `HyperFocus OS` that helps with pitch, build, scope, agent routing, risk control, and submission readiness.

- `docs/war-room/01_MASTER_BRIEF.md`

- `docs/war-room/04_AGENT_COMMAND.md`
- `docs/war-room/05_BUILD_SYSTEM.md`
- `docs/war-room/06_SPRINT_PLAN.md`
- `docs/war-room/07_RISK_SECURITY_QA.md`
- `docs/war-room/08_SUBMISSION_CHECKLIST.md`
- `docs/WAR_ROOM_LATEST.md`

## Product Position
- Lead with `HyperFocus OS`
- Use `MVP Relay` as the hidden workflow engine
- Keep `Founder NOC` as a later roadmap layer

## Design Rules
- Keep docs short and operational
- Avoid duplicate truths
- Optimize for fast agent handoffs
- Protect the hero flow from scope sprawl
- Keep language safe, non-clinical, and pitch-ready

## Approval
This design was approved before writing the pack.
