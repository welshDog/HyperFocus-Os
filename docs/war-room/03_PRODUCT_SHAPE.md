# HyperFocus OS War Room: Product Shape

## Product Stack
- Public product: `HyperFocus OS`
- Internal engine: `MVP Relay`
- Future expansion: `Founder NOC`

## V1 Hero Flow
1. User dumps everything pulling attention right now.
2. System groups and ranks it into `Now`, `Next`, and `Later`.
3. User starts one focus sprint.
4. User can tap `I'm stuck` to shrink the task again.
5. System ends with a visible win and calm recap.

## Core Screens
- Capture Screen
  - One large input box
  - One supportive line of copy
  - One primary action
- Prioritized Plan Screen
  - `Now`, `Next`, `Later`
  - Short reasons for ordering
  - One primary action: `Start Now`
- Focus Sprint Screen
  - Single task headline
  - Tiny checklist or steps
  - Quiet timer if needed
  - Rescue action: `I'm stuck`
- Win Screen
  - One completed step
  - One suggested next action
  - Calm recap

## Product Rules
- One obvious primary action per screen
- No noisy dashboard on first load
- No long onboarding before the user gets value
- Show visible reasoning without overwhelming detail
- Support the user; do not score or shame them

## In Scope For V1
- Brain dump capture
- AI triage into `Now`, `Next`, and `Later`
- Focus sprint creation
- Rescue flow for stuck moments
- Calm recap and next-step suggestion
- Seeded demo mode

## Out Of Scope For V1
- Complex habit systems
- Team collaboration
- Deep calendar/email integrations
- Heavy analytics
- Clinical claims
- Full Founder NOC surface

## MVP Relay's Job
- Translate messy user input into structured execution states
- Turn outputs into steps, tasks, and handoff-ready data
- Stay mostly invisible in the first public pitch
- Give the system a strong expansion path later

## Founder NOC's Job Later
- Add founder command center features
- Surface operating risks and priorities
- Provide live business context
- Expand HyperFocus OS into a wider execution platform

## Design Principle
HyperFocus OS should feel like relief, not management software.

## Command Choice
- Demo `HyperFocus OS`
- Build the `MVP Relay` engine under the hood
- Mention `Founder NOC` only as roadmap gravity
